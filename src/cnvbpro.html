CNVb Profile

Objective: Query the CNVb profile for each accession.

  1. 1. Select "CNVb Profile" from the main menu and enter or select the accession name.
  2. 2. There are two type of visualization mode, including chromosome profile and QR-code-like fingerprint.
  3. 3. Chromosome profile: Colored regions represent the presented markers.
  4. 4. QR-code-like: Digital present-absent status of 1,240 CNVb markers.
